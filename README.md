### Instructions

#### Frontend
- Run `cd client`
- Run `npm start` or `npm run lint:fix` or `npm run lint` for relevant commands.
#### Backend
- Run `cd server`
- Run `npm start`
- In dev, the frontend will be on localhost:8080 and backend will be '/api/(route)'
